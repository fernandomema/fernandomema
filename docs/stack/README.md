This document provides an overview of the Stack libaries and how to use them in your projects.

# Libraries
- **Notifications**: Sileo. A powerful and customizable notification library.
    - [GitHub Repository](https://github.com/hiaaryan/sileo)
    - [Svelte clone](https://github.com/alexisgvrcia/svelte-sileo)
- **Styling**: Tailwind CSS. A utility-first CSS framework for rapid UI development.
- **Icon system**: Iconify. Tailwind 4 plugin for Iconify icons.
- **Database system**: PrismaDB
- **Auth system**: better-auth. A simple and secure authentication library.
  - JWT based auth with refresh token.
- **Video player**: vidstack. A customizable video player library.
    - Integrations:
        - hls.js: A JavaScript library for playing HLS (HTTP Live Streaming) videos.
        - dash.js: A JavaScript library for playing DASH (Dynamic Adaptive Streaming over HTTP)
- **Haptics**: WebHaptics. A library for adding haptic feedback to web applications.
    - [GitHub Repository](https://github.com/lochie/web-haptics)
- **Fluid number swapping**: number-flow. A library for creating smooth and fluid number transitions.
    - [GitHub Repository](https://github.com/barvian/number-flow)
- **Auto animate**: A library for creating smooth animations with minimal code.
    - [GitHub Repository](https://auto-animate.formkit.com)
