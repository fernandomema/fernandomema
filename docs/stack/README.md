- <img src="https://api.iconify.design/simple-icons:prisma.svg" alt="database" width="18" height="18" style="vertical-align: middle;"> **Database system**: PrismaDB

    - JWT based auth with refresh token.
- <img src="https://api.iconify.design/mdi:translate.svg?color=%236366F1" alt="translate" width="18" height="18" style="vertical-align: middle;"> **Localization**: [Wuchale](https://wuchale.dev/intro/start/)
- <img src="https://api.iconify.design/mdi:play-box-outline.svg?color=%23EF4444" alt="video" width="18" height="18" style="vertical-align: middle;"> **Video player**: vidstack. A customizable video player library.
    - Integrations:
        - <img src="https://api.iconify.design/mdi:file-video-outline.svg?color=%23EF4444" alt="hls" width="18" height="18" style="vertical-align: middle;"> hls.js: For playing HLS (HTTP Live Streaming) videos.
        - <img src="https://api.iconify.design/mdi:file-video-outline.svg?color=%23EF4444" alt="dash" width="18" height="18" style="vertical-align: middle;"> dash.js: For playing DASH (Dynamic Adaptive Streaming over HTTP)
- <img src="https://api.iconify.design/mdi:vibrate.svg?color=%23A855F7" alt="haptic" width="18" height="18" style="vertical-align: middle;"> **Haptics**: WebHaptics. A library for adding haptic feedback to web applications.
    - <img src="https://api.iconify.design/mdi:github.svg?color=%23181717" alt="GitHub" width="18" height="18" style="vertical-align: middle;"> [GitHub Repository](https://github.com/lochie/web-haptics)
- <img src="https://api.iconify.design/mdi:numeric.svg?color=%2314B8A6" alt="numbers" width="18" height="18" style="vertical-align: middle;"> **Fluid number swapping**: number-flow. A library for creating smooth and fluid number transitions.
    - <img src="https://api.iconify.design/mdi:github.svg?color=%23181717" alt="GitHub" width="18" height="18" style="vertical-align: middle;"> [GitHub Repository](https://github.com/barvian/number-flow)
- <img src="https://api.iconify.design/mdi:motion-outline.svg?color=%23EC4899" alt="motion" width="18" height="18" style="vertical-align: middle;"> **Auto animate**: A library for creating smooth animations with minimal code.
    - <img src="https://api.iconify.design/mdi:github.svg?color=%23181717" alt="GitHub" width="18" height="18" style="vertical-align: middle;"> [GitHub Repository](https://auto-animate.formkit.com)
